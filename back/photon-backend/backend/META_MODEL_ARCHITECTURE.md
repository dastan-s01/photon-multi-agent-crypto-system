# Архитектура мета-модели для торговли

## 🏗️ Двухслойная архитектура

### Слой 1: Базовые модели

Обучаются для каждой монеты:
- **RandomForest** - консервативная модель с меньшим риском переобучения
- **GradientBoosting** - агрессивная модель с консервативными параметрами для крипты
- **XGBoost** (если доступен) - продвинутый бустинг

**Параметры GradientBoosting (консервативные для крипты):**
```python
learning_rate=0.05      # Маленький learning_rate (не 0.1!)
max_depth=4           # Неглубокие деревья (не 5!)
min_samples_split=10   # Больше samples для разделения
min_samples_leaf=5     # Больше samples в листе
subsample=0.8         # Стохастический бустинг
max_features=0.7      # Ограничение признаков
```

### Слой 2: Мета-уровень

**1. Определение режима рынка:**
- **trend** - тренд (SMA10 > SMA20, расстояние > 1.5%)
- **flat** - флэт (низкая волатильность, боковое движение)
- **volatile** - высокая волатильность/паника (volatility > 3% или экстремальный RSI)

**2. Статический выбор по монете:**
```python
BTCUSDT → RandomForest (показал +12.09%)
SOLUSDT → GradientBoosting (показал +5.69%)
ETHUSDT → RandomForest
ADAUSDT → GradientBoosting
XRPUSDT → GradientBoosting
DOGEUSDT → GradientBoosting
```

**3. Взвешенное голосование:**
- Основная модель получает вес 0.7
- Остальные модели получают вес 0.3 / (n-1)
- Веса могут динамически меняться на основе производительности

## 🔄 Как работает мета-модель

1. **Обучение:** Для каждой монеты обучаются все базовые модели (RF, GB, XGBoost)
2. **Определение режима:** Анализируются последние 20 свечей для определения режима рынка
3. **Выбор модели:** 
   - Статический выбор по монете (BTC → RF, SOL → GB)
   - Динамический выбор по режиму (если есть статистика)
4. **Ансамбль:** Взвешенное голосование всех моделей
5. **Предсказание:** Финальное решение с учетом уверенности

## 🎯 Преимущества

1. **Адаптивность:** Разные модели для разных монет
2. **Устойчивость:** Ансамбль снижает риск ошибок одной модели
3. **Режим-свитчинг:** Учет состояния рынка (тренд/флэт/паника)
4. **Консервативность:** Параметры GB настроены для крипты (меньше переобучение)
